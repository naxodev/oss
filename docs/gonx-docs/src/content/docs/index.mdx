## Key features

<div class="features-grid">
  <div class="feature-item">
    <h3 class="feature-title">Inferred tasks</h3>
    <p class="feature-description">
      gonx detects each <code>go.mod</code> and infers build, serve, test, lint,
      tidy, and generate targets automatically — no hand-written project.json
      files.
    </p>
  </div>

{' '}

{' '}

<div class="feature-item">
  <h3 class="feature-title">Project graph from static analysis</h3>
  <p class="feature-description">
    A tree-sitter parser reads Go imports and <code>go.mod</code> files to build
    Nx project dependencies.
  </p>
</div>

{' '}

  <div class="feature-item">
    <h3 class="feature-title">Generators</h3>
    <p class="feature-description">
      Scaffold applications, libraries, and Go Blueprint projects with web
      frameworks and database drivers.
    </p>
  </div>
</div>
